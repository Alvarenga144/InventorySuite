REQUIREMENTS

Se solicita el desarrollo C# en .Net Core 8 de una aplicación que consuma una API que tenga la funcionabilidad de inventario de productos con una tabla con los campos: (idpro, producto y precio), adicional una API para el detalle de ventas realizadas con una tabla de encabezado de venta con los campos: (idventa, fecha, vendedor, total) y otra tabla con el detalle de ventas campos: (idde, fecha, idventa, idpro, cantidad, precio, iva, total) los datos deben estar almacenados en base de datos usando el motor SQL Server.

Para esta prueba se requiere lo siguiente:
- La aplicación debe contar con inicio de sesión (usuario y contraseña) previamente registrado.
- El usuario operador tendrá solo la opción de realizar ventas.
- Desarrollo de una aplicación web que consuma API Rest de productos, esta debe realizar mantenimiento (CRUD)
- La aplicación web debe realizar la búsqueda de productos por código y agregar en detalle de venta a realizar, esta debe realizar los cálculos en pantalla.
- El sistema debe guardar los datos de la venta haciendo uso del API Rest.
- Creación de tabla productos, ventas y detalleventas, hacer uso de procedimientos almacenados.
- La aplicación debe mostrar notificaciones de la operación realizada y contar con manejo de errores.
- Generar un reporte en PDF y excel de ventas realizadas.
- Se solicita el desarrollo C# en .Net Core

NOTAS
- Contraseña de usuario debe estar encriptada (mostrar en base de datos)
- Al eliminar un producto debe mostrar una alerta si está seguro de la operación de dicho registro.
- Hacer uso del patrón MVC
- Hacer uso de JavaScript si se requiere.
- Hacer uso de Entity Framework
